# 📄KeyWord based Resume Screener

A sleek, FastAPI-based web application that evaluates resumes against specific industry domains using keyword extraction and scoring logic.

## 🌌 Features
* **Dynamic Scoring:** Evaluates resumes based on "Great," "Good," and "Medium" keyword matches.
* **Domain Specific:** Tailored analysis for Data Science, Software Development, Marketing, and Finance.
* **Multi-Format Support:** Processes both `.docx` and `.pdf` files.
* **Modern UI:** Dark purple sky theme with a clean, responsive interface.

## 🛠️ Tech Stack
* **Backend:** [FastAPI](https://fastapi.tiangolo.com/) (Python)
* **Frontend:** HTML5, CSS3, JavaScript
* **File Parsing:** `python-docx`, `PyPDF2`
* **Server:** `uvicorn`
